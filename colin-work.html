Featured Project

Building things from the ground up.

Research Project · BC Govindasamy Lab · 2023–Present

Remote-Controlled
Gantry System

Designed and built a fully automated gantry system to collect photodetector data across a precise spatial grid — enabling indoor positioning research that previously had to be done by hand.

The problem: Collecting accurate indoor positioning data required moving a photodetector to hundreds of specific coordinates. Manual placement took hours and introduced human error.

The solution: A custom-built gantry controlled by a Raspberry Pi and Arduino, integrated with open-source APIs to automate full grid sweeps. The system reduced data collection time dramatically and enabled machine learning models to be trained on far larger, more consistent datasets.

CAD / Fusion 360 Raspberry Pi Arduino Python Soldering ML
